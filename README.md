# ColorChooser
Небольшой скрипт на Python который позволяет выбирать цвета из палитры и просматривать их в HEX формате
## Installation
Необходимо:
- python 3.8
- tkinter 
```sh
git clone https://github.com/DeveloperDmitryKolyadin/colorchooser.git
cd colorchooser
python main.py
```
